# 通話受付管理のコマンドレット

 

_**トピックの最終更新日:** 2016-12-08_

通話受付管理 (CAC) を使用すると、リアルタイムの音声またはビデオ セッションを許容範囲内の品質で構築するのに十分なネットワーク帯域幅があるかどうかを判定できます。ネットワーク、サイト、サブネット、およびそれらの間の対話に制限と設定を構成することで、CAC を管理します。
