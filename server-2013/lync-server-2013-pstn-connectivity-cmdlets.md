# PSTN 接続のコマンドレット

 

_**トピックの最終更新日:** 2016-12-08_

エンタープライズ VoIP を使用すると、サービスの品質 (QoS) を低下させずに公衆交換電話網 (PSTN) との間で通話できる設定が可能になります。Lync Server 管理シェルから使用できるコマンドレットで、これらの設定を構成できます。
