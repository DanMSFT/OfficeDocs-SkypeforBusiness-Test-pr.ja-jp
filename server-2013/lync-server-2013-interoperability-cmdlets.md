# 相互運用性のコマンドレット

 

_**トピックの最終更新日:** 2012-06-27_

Microsoft Lync Server 2013 と、Microsoft Exchange Server 2013 など他のサーバー製品との間でサーバー間認証および承認を構成するには、相互運用コマンドレットを使用します。サーバー間認証および承認を使用すると、これらのサーバーがデータをシームレスに交換したり、共有したりできます。

## 相互運用コマンドレット

以下は、Microsoft Lync Server 2013 と他のサーバー製品間における相互運用性の構成と管理に直接関係するコマンドレットの一覧です。

**相互運用コマンドレット**

  - [Get-CsOAuthConfiguration](get-csoauthconfiguration.md)

  - [Set-CsOAuthConfiguration](set-csoauthconfiguration.md)

  - [Get-CsOAuthServer](get-csoauthserver.md)

  - [New-CsOAuthServer](new-csoauthserver.md)

  - [Remove-CsOAuthServer](remove-csoauthserver.md)

  - [Set-CsOAuthServer](set-csoauthserver.md)

  - [Get-CsPartnerApplication](get-cspartnerapplication.md)

  - [New-CsPartnerApplication](new-cspartnerapplication.md)

  - [Remove-CsPartnerApplication](remove-cspartnerapplication.md)

  - [Set-CsPartnerApplication](set-cspartnerapplication.md)
