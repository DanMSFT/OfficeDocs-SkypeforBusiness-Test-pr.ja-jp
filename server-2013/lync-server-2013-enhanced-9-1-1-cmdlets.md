# Enhanced 9-1-1 のコマンドレット

 

_**トピックの最終更新日:** 2016-12-08_

Microsoft Lync Server 2013 には、エンタープライズ VoIP ソリューションの Enhanced 9-1-1 (E9-1-1) を実装して管理するためのコマンドレットが含まれます。これらのコマンドレットを使用して、物理アドレスに接続ポイントをマッピングし、エンタープライズ VoIP ユーザーが正常に緊急電話を完了して、緊急サービス プロバイダーに場所を自動送信するために必要な設定を構成します。Lync Server コントロール パネルからは E9-1-1 を構成できません。コマンドレットを使用する必要があります。

## Enhanced 9-1-1 のコマンドレット
