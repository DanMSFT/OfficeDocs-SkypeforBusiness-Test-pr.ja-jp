# Lync Server 2013 での証明書および認証のコマンドレット

 

_**トピックの最終更新日:** 2016-12-08_

証明書と認証のコマンドレットは、さまざまな範囲のタスクに使用できます。たとえば、サーバーおよびクライアント証明書の管理、ユーザーの PIN (暗証番号) の管理、インターネット インフォメーション サービスで使用する SIP ドメインと Kerberos アカウントの管理などです。
